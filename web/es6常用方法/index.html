简介

慢慢成为es6的熟练小能手。

数组的方法

map

遍历原数组所有item，通过返回修改原来新item，组成新数组。

filter

遍历原数组所有item，通过返回true或者发false，过滤原数组的item组成新数组。

forEach

遍历原数组，不能break中途退出，可以continue，return只支持退出循环（无视return的值）。

some

遍历数组，当有一真时即为真。（相当于判断数组中是具有否符合要求item）

every

遍历数组，当有一假时即为假。（相当于判断数组中的所有item是否符合要求）
