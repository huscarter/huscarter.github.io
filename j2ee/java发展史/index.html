javaеҸ‘еұ•еҸІ

2016-09-12 | j2ee

1991е№ҙJames GoslingеҚҡеЈ«йўҶеҜјзҡ„Green ProjectејҖе§ӢеҗҜеҠЁпјҢзӣ®зҡ„жҳҜејҖеҸ‘дёҖз§ҚиғҪеӨҹеңЁеҗ„з§Қж¶Ҳиҙ№жҖ§з”өеӯҗдә§е“ҒдёҠиҝҗиЎҢзҡ„зЁӢеәҸжһ¶жһ„гҖӮиҝҷдёӘ
и®ЎеҲ’зҡ„дә§е“Ғе°ұжҳҜjavaзҡ„еүҚиә«OakпјҲж©Ўж ‘пјүгҖӮ

1995е№ҙдә’иҒ”зҪ‘жҪ®жөҒе…ҙиө·пјҢOakз»ҲдәҺжүҫеҲ°йҖӮеҗҲиҮӘе·ұеҸ‘еұ•зҡ„еёӮеңәпјҢиө·жӯ»еӣһз”ҹе№¶иң•еҸҳжҲҗjavaгҖӮ5жңҲ23ж—ҘеңЁSunWorldеӨ§дјҡдёҠжӯЈејҸеҸ‘еёғдәҶJDK1.0
зүҲжң¬пјҢеҸЈеҸ·дёәпјҡвҖңWrite once,Run anywhereвҖқ,иҮіжӯӨjavaиҜһз”ҹпјҒ

1997е№ҙSunеҸ‘еёғдәҶJDK1.1пјҢjavaжҠҖжңҜзҡ„дёҖдәӣжңҖеҹәзЎҖзҡ„ж”Ҝж’‘зӮ№пјҲеҰӮJDBCпјҢJARж–Үд»¶ж јејҸпјҢJavaBeanпјҢRMIзӯүпјүйғҪжҳҜжҳҜеңЁjava1.1зүҲжң¬
дёӯеҸ‘еёғзҡ„гҖӮжӯӨеӨ–javaиҜӯжі•д№ҹжңүдәҶдёҖдәӣеҸ‘еұ•пјҢжҜ”еҰӮеҠ е…ҘдәҶеҶ…йғЁзұ»пјҲinner classпјүе’ҢеҸҚе°„пјҲreflectionпјүгҖӮ

1998е№ҙJDKиҝҺжқҘдәҶдёҖдёӘзұ»дјјйҮҢзЁӢзў‘зҡ„зүҲжң¬JDK1.2пјҢSunеңЁиҝҷдёӘзүҲжң¬дёӯжҠҠjavaжҠҖжңҜдҪ“зі»жӢҶеҲҶдёә3дёӘж–№еҗ‘пјҢ
еҲҶеҲ«жҳҜйқўеҗ‘жЎҢйқўеә”з”ЁејҖеҸ‘зҡ„J2SE(java 2 platform,standard edition),йқўеҗ‘з§»еҠЁз«ҜејҖеҸ‘зҡ„J2ME(java 2 platform,Micro edition)е’Ң
йқўзӣёеҗ‘дјҒдёҡзә§ејҖеҸ‘зҡ„J2EE(java 2 platform,enterprise edition).иҝҷдёӘзүҲжң¬дёӯеҮәзҺ°зҡ„д»ЈиЎЁжҠҖжңҜйқһеёёеӨҡпјҢжҜ”еҰӮEJBгҖҒjava plugin-inгҖҒ
java IDLгҖҒswingзӯүгҖӮе№¶дё”еңЁиҝҷдёӘзүҲжң¬дёӯjavaиҷҡжӢҹжңәз¬¬дёҖж¬ЎеҶ…зҪ®дәҶJITпјҲjust in timeпјүзј–иҜ‘еҷЁпјҢJDK1.2жӣҫе№¶еӯҳиҝҮ3дёӘиҷҡжӢҹжңәпјҢ
classic VMгҖҒhotspot VMе’Ңexact VMпјҢhotspotе’Ңexact VMйғҪеҶ…зҪ®дәҶJITпјҢиҖҢclassic VMеҸӘиғҪд»ҘеӨ–жҢӮзҡ„еҪўејҸдҪҝз”ЁJITгҖӮ

2000е№ҙSunеҸ‘еёғдәҶJDK1.3гҖӮJDK1.3зӣёеҜ№дәҺJDK1.2дё»иҰҒзҡ„ж”№иҝӣиЎЁзҺ°еңЁдёҖдәӣзұ»еә“дёҠпјҲеҰӮж•°еӯҰиҝҗз®—е’Ңж–°зҡ„Timer APIзӯүпјүпјҢиҝҷдёӘзүҲжң¬еҜ№
java 2DеҒҡдәҶеҫҲеӨҡж”№иҝӣпјҢжҸҗдҫӣдәҶжңҖж–°зҡ„java 2D APIпјҢе№¶ж·»еҠ дәҶJavaSoundзұ»еә“гҖӮиҮӘJDK1.3ејҖе§ӢпјҢSunз»ҙжҢҒдәҶдёҖдёӘд№ жғҜпјҡеӨ§зәҰжҜҸйҡ”
дёӨе№ҙеҸ‘еёғдёҖдёӘJDKдё»зүҲжң¬гҖӮ

2002е№ҙ2жңҲ13ж—ҘпјҢJDK1.4еҸ‘еёғгҖӮJDK1.4жҳҜjavaзңҹжӯЈиө°еҗ‘жҲҗзҶҹзҡ„дёҖдёӘзүҲжң¬пјҢе“ӘжҖ•еҚҒеӨҡе№ҙеҗҺзҡ„д»ҠеӨ©пјҢд»Қз„¶еҗҺи®ёеӨҡдё»жөҒзҡ„еә”з”Ё
пјҲspringгҖҒhibernateгҖҒstrutsзӯүпјүиғҪзӣҙжҺҘиҝҗиЎҢеңЁJDK1.4д№ӢдёҠгҖӮJDK1.4еҸ‘еёғдәҶеҫҲеӨҡж–°зҡ„жҠҖжңҜзү№жҖ§пјҢеҰӮжӯЈеҲҷиЎЁиҫҫејҸпјҢ
ејӮеёёй“ҫгҖҒNIOпјҲйқһйҳ»еЎһзј“еӯҳпјүе’Ңж—Ҙеҝ—гҖҒXMLи§ЈжһҗзӯүгҖӮ

2004е№ҙ9жңҲ30ж—ҘпјҢJDK1.5еҸ‘еёғгҖӮд»ҺJDK1.2д»ҘжқҘпјҢjavaеңЁиҜӯжі•еұӮйқўдёҠзҡ„еҸҳеҢ–дёҖзӣҙеҫҲе°ҸпјҢиҖҢJDK1.5еңЁjavaиҜӯжі•жҳ“з”ЁжҖ§дёҠеҒҡдәҶйқһеёёеӨ§зҡ„ж”№иҝӣгҖӮ
дҫӢеҰӮиҮӘеҠЁе°ҒиЈ…гҖҒиҢғеһӢгҖҒеҠЁжҖҒжіЁи§ЈгҖҒжһҡдёҫгҖҒеҸҜеҸҳй•ҝеҸӮж•°е’ҢйҒҚеҺҶеҫӘзҺҜforeachзӯүгҖӮеҸҰеӨ–JDK1.5жҳҜе®ҳж–№з”іжҳҺеҸҜд»Ҙж”ҜжҢҒwindows 9xе№іеҸ°зҡ„жңҖеҗҺ
дёҖдёӘJDKзүҲжң¬гҖӮ

2006е№ҙ12жңҲ11ж—ҘпјҢJDK1.6еҸ‘еёғгҖӮеңЁиҝҷдёӘзүҲжң¬дёӯSunз»“жқҹдәҶд»ҺJDK1.2д»ҘжқҘзҡ„J2EEгҖҒJ2SEе’ҢJ2MEзҡ„е‘ҪеҗҚж–№ејҸпјҢж”№з”ЁJava EE6гҖҒJava SE6
е’ҢJava ME6зҡ„е‘ҪеҗҚж–№ејҸгҖӮJDK1.6зҡ„ж”№иҝӣеҢ…жӢ¬пјҡжҸҗдҫӣеҠЁжҖҒиҜӯиЁҖж”ҜжҢҒгҖҒжҸҗдҫӣзј–иҜ‘APIе’Ңеҫ®еһӢhttpжңҚеҠЎеҷЁAPIзӯүпјҢеҗҢж—¶еҜ№javaиҷҡжӢҹжңәеҒҡдәҶеӨ§йҮҸзҡ„
ж”№иҝӣгҖӮеңЁеҪ“е№ҙзҡ„JavaOneеӨ§дјҡдёҠSunе…¬еҸёе®Јеёғе°ҶjavaејҖжәҗпјҢе№¶еңЁйҡҸеҗҺзҡ„дёҖе№ҙеӨҡж—¶й—ҙеҶ…йҷҶз»ӯе°ҶJDKзҡ„еҗ„дёӘйғЁеҲҶеңЁGPL v2зҡ„еҚҸи®®дёӢе…¬ејҖдәҶжәҗз ҒпјҢ
еҗҢж—¶е»әз«ӢдәҶOpenJDKз»„з»ҮеҜ№иҝҷдәӣжәҗз ҒиҝӣиЎҢзӢ¬з«Ӣз®ЎзҗҶгҖӮ

2009е№ҙ2жңҲ19ж—ҘпјҢJDK1.7е®ҢжҲҗдәҶе…¶з¬¬дёҖдёӘйҮҢзЁӢзў‘зүҲжң¬гҖӮж №жҚ®JDK1.7зҡ„еҠҹиғҪи§„еҲ’пјҢдёҖе…ұи®ҫзҪ®дәҶ10дёӘйҮҢзЁӢзў‘пјҢжңҖеҗҺдёҖдёӘеҺҹи®ЎеҲ’дәҺ2010е№ҙ9жңҲ9ж—Ҙ
з»“жқҹпјҢдҪҶжҳҜз”ұдәҺеҗ„з§ҚеҺҹеӣ JDKжңҖз»Ҳж— жі•жҢүи®ЎеҲ’е®ҢжҲҗгҖӮеңЁJDK1.7еҜ№ARMжҢҮд»ӨйӣҶжһ¶жһ„жҸҗдҫӣдәҶж”ҜжҢҒпјҢиҮіжӯӨпјҢJDKеҸҜд»ҘиҝҗиЎҢеңЁWindowsпјҲдёҚеҗ«Windows 9xпјүгҖҒ
LinuxгҖҒSolarisе’ҢMac OSе№іеҸ°дёҠпјҢж”ҜжҢҒARMгҖҒx86гҖҒx64е’ҢSparcжҢҮд»ӨйӣҶгҖӮ

2009е№ҙ4жңҲ20ж—ҘгҖӮOracleд»Ҙ74дәҝзҫҺе…ғж”¶иҙӯSunе…¬еҸёпјҢjavaе•Ҷж Үд»ҺжӯӨеҪ’дёәOracleжүҖжңүгҖӮ

жіЁпјҡжң¬ж–ҮеҶ…е®№еј•з”ЁдәҶе‘Ёеҝ—жҳҺзҡ„гҖҠж·ұе…ҘзҗҶи§ЈJavaиҷҡжӢҹжңәгҖӢгҖӮ
