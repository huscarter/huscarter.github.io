移动开发设计规范

2020-05-08 | computicsandroid

系统颜色

主题颜色

  1. 主题主色调

  2. 背景颜色

  3. 阴影颜色

  4. 蒙层颜色，用于展示该item已选中不可点击,[B3：70]

  5. 弹框透明底色

字体颜色

  1. 标题、或者是文本主题颜色，用于一级文字。如导航栏标题，产品信息，个人中心，弹窗状等

  2. 二级标题颜色，用于二级文字信息。如导航栏二级标题，产品辅助信息，分享，弹窗副文本内容等

  3. 文本内容颜色，用于大面积描述性文本。如标题下面的正文。

  4. 说明类型文字颜色，用于说明、解释性文字信息。如产品交易记录时间显示，产品下介绍内容等

  5. 输入框说明类型文字颜色，用于登录/注册输入框提示信息。

  6. 不可用的字体颜色

  7. 项目警示性信息颜色

按钮颜色

  1. 正常按钮颜色

  2. 按钮按下的颜色（一般系统有水波纹效果）

  3. 不可点击按钮的颜色

分割线颜色

  1. 正常分割线颜色

主题的红、黄、蓝、绿、橘

  1. 红

  2. 黄

  3. 蓝

  4. 绿

  5. 橘

其他

  1. 提示信息背景色，比如用于断网提示背景

系统尺寸

主题尺寸

  1. 标题栏高度

  2. 状态栏高度（系统不可更改）

  3. 标题栏菜单宽度和高度（一般推荐采用标题栏高的正方形）

  4. 边距（内容到屏幕的距离）

  5. 卡片、条目之间的上下间距

组件尺寸

  1. 按钮的高度

  2. 输入框的高度

  3. 条目的高度

  4. 搜索框高度

  5. 分割线的高度

文字尺寸

  1. 超大文本文字尺寸

  2. 大文本文字尺寸

  3. 标题文字尺寸

  4. 普通按钮字体

  5. 二级标题文字尺寸

  6. 文本文字尺寸

  7. 说明文字尺寸

  8. 输入框提示文字尺寸

  9. 小文本文字尺寸，比如红点消息个数

圆角尺寸

  1. 卡片圆角

  2. 按钮圆角

  3. 弹框圆角
